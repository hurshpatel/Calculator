A Little Background

Well you can probably guess what this app would do, and yes there are numerous calculator applications on the market. What makes this one different? To be honest, nothing right now.. This was the first app I've ever made using Swift and a completely beginner project. But I'm working on updating it to include a graphing function right now, which implements a split view for the iPhone 6 Plus and iPads. You'll find screenshots below and I'll explain the functionalities of the app, and maybe a little bit of code which I found interesting.

The Meat and Potatoes

Starting with the interface, it is a standard interface with buttons and a "screen" to show the input and computations. The operations include square root, variables, sin/cos, +/-, e, pi, %, etc. There is another, smaller "screen" on top which shows the stack of inputs from the user. This stack is formatted to look like an actual equation, which makes for better readability. This is for the user to see what they have entered.
